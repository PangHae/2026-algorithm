# 신한투자증권 ICT 공채 코딩테스트 준비

> 2022~2024년 후기 기반 정리  
> 플랫폼: 프로그래머스 | 문제 수: 3문제 내외 | 목표 난이도: 실버3 ~ 골드4

---

## 📋 시험 개요

| 항목 | 내용 |
|------|------|
| 플랫폼 | 프로그래머스 |
| 문제 수 | 3문제 (2024 하반기 기준) |
| 난이도 범위 | 브론즈 ~ 골드5 |
| 목표 합격 기준 | 3문제 중 2문제 이상 완벽 풀이 |
| 특이사항 | 금융권치고 생각보다 어렵다는 후기 다수 |

### 문제 구성 패턴

- **1번** — 문자열 처리 or 간단한 구현 (실버 수준). 20분 이내 클리어 목표
- **2번** — BFS/DFS 혹은 그리디 (실버~골드5 수준). 40분 이내 목표
- **3번** — 구현+알고리즘 복합 or DP (골드 수준). 부분 점수 전략 유효

## 📌 핵심 전략

- **금융권이라고 방심 금지** — 생각보다 어렵다는 후기가 많음. 실버 탄탄 + 골드5 풀이 경험 필요
- **1번 문제**는 문자열 처리 or 간단한 구현. 빠르게 20분 안에 클리어하는 게 목표
- **2번 문제**는 BFS/DFS 혹은 그리디. 실버~골드5 수준. 40분 내 목표
- **3번 문제**는 구현+알고리즘 복합 or DP. 어려운 편. 부분 점수라도 챙기는 전략 유효
- **엣지케이스**를 항상 체크할 것. 쉬운 문제일수록 엣지케이스에서 감점됨
- 백준에서 유형 잡은 뒤 **프로그래머스 문제로 전환**해서 플랫폼 감각 익히기

## 📅 10일 단기 학습 플랜 (3월 30일 마감 기준)

| 기간 | 주제 | 세부 내용 |
|------|------|-----------|
| 1~2일 | 문자열 + 스택/큐 기초 | 괄호 검사, 단어 뒤집기, 후위표기식. 구현 속도와 파싱 실수 없애기 |
| 3~4일 | BFS/DFS 격자 탐색 집중 | 미로 탐색 → 토마토 → 연구소 순서로 풀고 코드 템플릿 손에 익히기 |
| 5~6일 | 구현/시뮬레이션 골드 연습 | 로봇 청소기, 뱀, 미세먼지. 조건 많은 문제를 꼼꼼하게 구현하는 연습 |
| 7~8일 | 그리디 + 완전탐색 + DP 핵심 | 회의실 배정, N-Queen, LIS/LCS, 배낭 문제. 유형별 1~2문제씩 패턴 정리 |
| 9~10일 | 모의 풀이 + 약점 보완 | 프로그래머스 모의고사 3문제 세트 시간 재고 풀기. 틀린 유형 집중 복습 |
